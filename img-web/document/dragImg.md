#拖拽文本框实现效果
1.记录 x 和 y 的坐标
2.停止拖拽时把文本框的 x 和  y 的坐标动态跟新为记录的 x 和 y的坐标

offsetLeft 位置含义：当前的位置与相对定位的距离 x
offsetTop  位置含义：当前的位置与相对定位的距离 y

e.clientX 位置含义：当前的位置（鼠标点击的点）与dom的距离 x

e.clientY 位置含义：当前的位置（鼠标点击的点）与dom的距离 y
